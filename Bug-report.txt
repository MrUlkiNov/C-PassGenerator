**Заголовок:** Неверная длина пароля при вводе нечислового значения

**ID:** BR-002

**Дата:** 2024-11-13

**Автор:** Самир Байбеков

**Описание:** При вводе нечислового значения в поле длины пароля и нажатии кнопки "Generate Password" возникает ошибка, так как приложение не может обработать некорректный ввод. Вместо обработки ошибки пользователь получает сообщение об ошибке, что нарушает логику работы интерфейса.

**Шаги для воспроизведения:**
1. Запустить приложение Greek Password Generator.
2. Ввести текстовое значение, например, "abc" в поле длины пароля.
3. Нажать кнопку "Generate Password".

**Ожидаемый результат:** Приложение не принимает нечисловой ввод и выдает сообщение о необходимости ввести корректное числовое значение длины пароля.

**Фактический результат:** Приложение показывает стандартное сообщение об ошибке, не давая пользователю возможности исправить некорректный ввод.

**Прикрепленные файлы:**
- Скриншот окна с сообщением об ошибке.

**Приоритет:** Средний

**Статус:** Открыт

**Дополнительная информация:**
- Версия приложения: 1.0.0
- Операционная система: Windows 10
- Используемый язык интерфейса: русский

**Рекомендации по исправлению:**
1. Ввести проверку на корректность ввода, которая разрешает только положительные числовые значения.
2. Добавить обработку исключений для нечислового ввода в поле длины пароля.
3. Улучшить сообщение об ошибке, чтобы информировать пользователя о допустимом значении.

**Заключение:** Исправление данного бага улучшит пользовательский опыт и поможет избежать ошибок, связанных с некорректным вводом в поле длины пароля.
